Pondiverse - How to take part in the Pondiverse

Using other people's tools

  1. Find a tool that's part of the Pondiverse. There is a list of tools on the create page.
  2. Click on a tool to go to its website.
  3. Make your thing in the tool. Keep fiddling until you're happy. It's like arranging flowers in a vase. This is going to be your creation.
  4. Click the Pondiverse button. It looks like this:
  5. Add a title and click submit.

Adding your tool to the Pondiverse

Import the Pondiverse script. One way to do this: 

import {addPondiverseButton} from 'https://www.pondiverse.com/script/pondiverse.js'

Then, write a function that returns the data Pondiverse wants. It should return a javascript object with attributes containing the type of your creation, the data associated with creating it, and a thumbnail image. You can make these anything you like.

Make this function an attribute of the global window object called getPondiverseCreation. It could look like this: 

window.getPondiverseCreation = () => {
    return {
        type: "splat",
        data: JSON.stringify({parameters: {a: 10, b: 10, c: 10},}),
        image: canvas?.toDataURL("image/png"),
    };
};

Finally, add the line 

addPondiverseButton();
anywhere in your script, after this function you just wrote. This is what adds the Pondiverse button to your page.

That's it! Have a nice day :)

Secret bonus section: adding your tool to this website

If you are happy using github, you can make a pull request to the participants.js file. This contains one big object with the names of all the types, along with links to the primary tool that makes them. This list gets put on the create page and with the titles in the gallery.

We wil probably merge it without checking.
